# AIArmada Filament Signals

Filament analytics UI for signals dashboards, reports, tracked properties, and alert management.

See [docs/](docs/) for installation, configuration, and usage.

## Safe remote interaction scanning

Remote page scans validate every DNS answer and use IP-pinned HTTP transport with redirects disabled, explicit connect/read timeouts, and HTML content-type checks. Mixed public/private answers and rebinding targets are rejected before any request is sent.
